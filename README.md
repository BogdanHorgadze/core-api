## Running the app

```bash
# database
$ npm run db:start

# backend
$ npm i
$ npm run dev
```

## Migrations
```bash
$ npm run typeorm:run-migrations 
$ npm run typeorm:generate-migration -name=Name 
$ npm run typeorm:create-migration -name=Name 
$ npm run typeorm:revert-migration 
```

## Nest

```bash
# create nest component
$ nest g resource NAME
```
